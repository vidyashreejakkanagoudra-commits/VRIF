#HTML Basic
Defines the basic structure of a webpage.
<!DOCTYPE html>
<html>
<body>
<h1>Hello World</h1>
</body>
</html>

HTML Elements
An element consists of a start tag, content, and an end tag.
<p>This is a paragraph</p>

HTML Attributes
Attributes provide extra information about elements.
<a href="https://www.w3schools.com">Visit</a>

HTML Headings
Used to define headings from <h1> to <h6>.
<h1>Main Heading</h1>
<h3>Sub Heading</h3>

HTML Paragraphs
Used to define text paragraphs.
<p>This is a paragraph.</p>

HTML Styles
Used to style HTML elements.
<p style="color:red;">Red Text</p>

HTML Formatting
Formats text appearance.
<b>Bold</b>
<i>Italic</i>
<u>Underline</u>

HTML Quotations
Used for quotes and citations.
<blockquote>This is a quote</blockquote>

HTML Comments
Used to add comments in HTML.
<!-- This is a comment -->

HTML Colors
Defines colors using names, RGB, or HEX.
<p style="color:blue;">Blue Text</p>

HTML CSS
CSS is used to style HTML elements.
<style>
p { color: green; }
</style>

HTML Links
Creates hyperlinks.
<a href="https://google.com">Google</a>

HTML Images
Displays images.
<img src="image.jpg" alt="Image">

HTML Favicon
Defines a website icon.
<link rel="icon" href="favicon.ico">

HTML Page Title
Sets the page title.
<title>My Webpage</title>

HTML Tables
Displays data in rows and columns.
<table>
<tr><th>Name</th><th>Age</th></tr>
<tr><td>Saniya</td><td>20</td></tr>
</table>

HTML Lists
Defines lists.
<ul>
  <li>Apple</li>
</ul>

HTML Block & Inline
Block elements start on a new line; inline do not.
<div>Block</div>
<span>Inline</span>

HTML Div
Defines a container.
<div style="background:lightblue;">Content</div>
HTML Classes
Used to style multiple elements.
<p class="text">Hello</p>

HTML Id
Used to uniquely identify an element.
<p id="para1">Unique</p>

HTML Buttons
Creates clickable buttons.
<button>Click Me</button>

HTML Iframes
Embeds another webpage.
<iframe src="https://w3schools.com"></iframe>

HTML JavaScript
Adds interactivity.
<button onclick="alert('Hi')">Click</button>

HTML File Paths
Defines file locations.
<img src="images/pic.jpg">

HTML Head
Contains metadata.
<head>
<title>Title</title>
</head>

HTML Layout
Defines page structure.
<header>Header</header>
<footer>Footer</footer>

HTML Responsive
Makes pages responsive.
<meta name="viewport" content="width=device-width, initial-scale=1.0">

HTML Computercode
Defines computer code.
<code>print("Hello")</code>

HTML Semantics
Uses meaningful elements.
<article>News</article>

HTML Style Guide
Best practices for clean code.
<p>Use lowercase tags</p>

HTML Entities
Displays reserved characters.
&lt; &gt; &copy;

HTML Symbols
Displays special symbols.
&trade; &euro;

HTML Emojis
Displays emojis using Unicode.
&#128512;

HTML Charsets
Defines character encoding.
<meta charset="UTF-8">

#HTML Forms
Used to collect user input.
<form>
  <input type="text" placeholder="Enter name">
</form>

HTML Form Attributes
Define how a form behaves.
<form action="submit.php" method="post">

HTML Form Elements
Elements inside a form that accept input.
<form>
  <input type="text">
  <select>
    <option>India</option>
  </select>
</form>

HTML Input Types
Different types of input fields.
<input type="password">
<input type="email">
<input type="date">

HTML Input Attributes
Provide extra control over input fields.
<input type="text" required maxlength="10">

Input Form Attributes
Attributes that belong to the <form> but apply to inputs.
<input type="text" form="myForm">
<form id="myForm"></form>

#HTML Graphics
HTML Canvas

<canvas width="200" height="100"></canvas>

HTML SVG
Used to draw scalable vector graphics.

Supports multimedia like audio and video.

HTML Video

<video width="320" controls>
  <source src="movie.mp4" type="video/mp4">
</video>

HTML Audio
Used to play audio files.
<audio controls>
  <source src="song.mp3" type="audio/mpeg">
</audio>

HTML Plug-ins
Used to embed external applications (deprecated).
<object data="file.pdf"></object>
